# whisperai 仕様書

## プロジェクトの目的

OpenAI Whisper の large モデルを使用して、動画の音声をオフラインで SRT ファイルとして書き出すソフトウェアを開発する。

## 実行方法

```
whisperai.py <動画ファイル> [言語コード]
```

- `言語コード` は `en`、`de`、`ja` などの ISO 639-1 短縮形
- 言語コードは字幕ありの動画にのみ有効（字幕なし動画では無視）

## プログラムの動作

### 字幕ありの動画

1. 動画ファイルの字幕トラックを確認してリストを出力
2. **言語コードを省略した場合**: 字幕トラック一覧を表示して終了。ユーザーが確認後、言語コードを指定して再実行する。
3. **言語コードを指定した場合**: 指定した言語の字幕トラックを選択し、英語に翻訳して SRT ファイルとして出力

### 字幕なしの動画

- 言語コードパラメータは無視する
- Whisper で音声を認識し、検出した言語でそのまま SRT ファイルを出力（翻訳なし）
- 例：日本語音声 → 日本語 SRT、英語音声 → 英語 SRT

## 出力ファイル

- ファイル名: `{入力ファイルのベース名}.{言語コード}.srt`
  - 例: `batman.mp4` → `batman.en.srt`
- 出力先: 入力動画と同じフォルダ

## モデル管理

- モデルは初回起動時に一度だけダウンロード
- 保存先: macOS の Application Support フォルダ（`~/Library/Application Support/whisperai/`）
- 使用モデル: Whisper large（最高精度）

## 進捗表示

- 処理中は `*` を順次追記する形式で進捗を表示する

## 技術スタック

- 言語: Python
- 音声認識・翻訳: openai-whisper（large モデル）
- 動画処理: ffmpeg / ffprobe
- 字幕翻訳: Whisper の translate タスク または Helsinki-NLP/opus-mt 等のオフライン翻訳モデル
