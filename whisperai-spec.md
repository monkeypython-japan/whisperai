# whisperai 仕様書

## プロジェクトの目的

OpenAI Whisper の large モデルを使用して、動画の音声をオフラインで SRT ファイルとして書き出すソフトウェアを開発する。

## 実行方法

```
whisperai.py <動画ファイル|フォルダ>... [言語コード] [オプション]
```

- 動画ファイル・フォルダは複数指定可。フォルダ指定時はサブフォルダも含め動画ファイル（mp4 / mkv / m4v / avi / mov / webm / ts / wmv）を名前順に一括処理する
- 位置引数は「存在するパス → 入力」「2〜3 文字の英字 → 言語コード」として解釈する（順不同）
- `言語コード` は `en`、`de`、`ja` などの ISO 639-1 短縮形
- 言語コードは字幕ありの動画にのみ有効（字幕なし動画では無視）
- `--translate[=言語]`: 字幕を指定言語に翻訳して出力（省略時は英語。字幕トラックありの場合のみ有効）。`--translate=ja` / `--translate ja` の両形式が使える
  - 英語への翻訳: Helsinki-NLP/opus-mt-mul-en
  - 英語以外（`--translate=ja` など）: NLLB-200（`facebook/nllb-200-distilled-600M`）。翻訳元言語は字幕トラックの言語タグから自動判定
- `--transcribe`: 字幕トラックを無視して Whisper で音声認識を強制実行
- `--help`: ヘルプを表示して終了

## プログラムの動作

### 字幕ありの動画

1. 動画ファイルの字幕トラックを確認してリストを出力
2. **言語コードを省略した場合**: 字幕トラック一覧を表示して終了。ユーザーが確認後、言語コードを指定して再実行する。
3. **言語コードを指定した場合**: 指定した言語の字幕トラックを抽出し、そのまま SRT ファイルとして出力
4. **`--translate` を付けた場合**: 抽出した字幕を英語に翻訳して SRT ファイルとして出力。`--translate=ja` のように翻訳先言語を指定できる（出力ファイルの言語コードは翻訳先言語）
5. **`--transcribe` を付けた場合**: 字幕トラックを無視して Whisper で音声認識を実行

### 字幕なしの動画 / --transcribe 指定時

- 言語コードパラメータは無視する
- Whisper で音声を認識し、検出した言語でそのまま SRT ファイルを出力（翻訳なし）
- 例：日本語音声 → 日本語 SRT、英語音声 → 英語 SRT

### 画像字幕（ビットマップ字幕）の扱い

- Blu-ray PGS（`hdmv_pgs_subtitle`）・DVD VobSub（`dvd_subtitle`）等の画像字幕はテキストとして抽出できない
- トラック一覧では「（画像字幕・抽出不可）」と表示する
- 同じ言語でテキスト字幕と画像字幕の両方がある場合はテキスト字幕を優先して選択する
- 画像字幕しかない言語を指定した場合はエラーとし、`--transcribe`（音声認識）の利用を案内する

## 出力ファイル

- ファイル名: `{入力ファイルのベース名}.{言語コード}.srt`
  - 例: `batman.mp4` → `batman.en.srt`
- 出力先: 入力動画と同じフォルダ

## 一括処理

- 複数ファイル・フォルダ指定時は 1 本ずつ順に処理する
- 1 本の失敗で停止せず続行し、最後に成功数と失敗ファイル一覧を表示する
- 1 本でも失敗があれば終了コード 1

## ノイズフィルター

以下のブロックを自動除去する（字幕トラック抽出・Whisper 出力の両方に適用）：

- 空ブロック
- クレジット・ノイズパターン（`Sous-titrage`、`subtitled by`、`OpenSubtitles` など）
- `©` 記号を含むブロック（配布者の透かし・Whisper ハルシネーションによる著作権表記）
- `transcript <名前>` 形式のブロック（Whisper ハルシネーションによるクレジット）
- 連続する完全一致テキスト（繰り返しハルシネーション）

## モデル管理

- モデルは初回起動時に一度だけ Hugging Face から自動ダウンロード
- 使用モデル: `mlx-community/whisper-large-v3-mlx`（Apple MLX 最適化版、M シリーズ GPU 使用）
- `word_timestamps=True` を有効化し、単語単位のタイムスタンプで各セグメントの終端を精確に決定する（30秒チャンク境界へのスナップを防止）

## 進捗表示

- 処理ステップの完了時に `*` を追記する形式で進捗を表示する
- 翻訳中は `翻訳中 [処理済み/総ブロック数]` 形式で進捗を表示する
- 音声認識中は mlx-whisper 標準のプログレスバーを表示する

## 技術スタック

- 言語: Python
- 音声認識: mlx-whisper（Apple MLX、M シリーズ GPU 活用）
- 動画処理: ffmpeg / ffprobe
- SRT パース: srt ライブラリ（BOM・改行コード・タイムコードの揺れに対応）
- CLI 引数処理: argparse
- テスト: pytest（`tests/` 配下、`python -m pytest tests/` で実行）
- 字幕翻訳: Helsinki-NLP/opus-mt-mul-en（英語向け）、facebook/nllb-200-distilled-600M（英語以外向け、いずれもオフライン翻訳モデル）
