# 0002. `--translate` を既定 OFF・明示指定制にする

## 状況

初期実装では、字幕トラックを抽出した場合に常に英語へ翻訳して出力していた。しかし、字幕をそのまま（原語のまま）欲しいケースも多く、翻訳が必要かどうかは動画・目的によって異なる。

## 決定

字幕トラック抽出時のデフォルト動作を「そのまま SRT 出力」とし、翻訳は `--translate`（英語）／`--translate=<言語>`（任意言語）を明示的に指定した場合のみ行う。

## 理由

- 既存の字幕をそのまま使いたい場合に、不要な翻訳処理（モデルロード・推論時間）を強制しない
- 翻訳先言語を選べるようにする（[[0004-翻訳モデルの言語別切り替え]]）ためには、そもそも「翻訳するかどうか」をオプトインにする必要があった

## 影響

- 出力ファイル名の言語コードは、翻訳しない場合は字幕トラックの言語、翻訳する場合は翻訳先言語になる
- 当初検討していた `--dump-srt`（翻訳前の生 SRT を別途保存するオプション）は、翻訳が既定 OFF になったことで存在意義が薄れたため廃止した
