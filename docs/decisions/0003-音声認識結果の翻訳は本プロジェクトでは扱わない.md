# 0003. 音声認識結果（字幕なし動画）への翻訳対応は本プロジェクトでは実装しない

## 状況

`--translate` は字幕トラックがある動画にのみ有効で、字幕なし動画では無視される。実際に「英語音声のみの動画から日本語字幕を作りたい」というユースケースが発生した（The Questor Tapes での検証時）。

拡張案としては、字幕なし動画 / `--transcribe` 指定時でも `--translate=<言語>` があれば Whisper 認識結果を続けて翻訳して出力する、というものが検討された。

## 決定

この拡張は whisperai 単体では実装しない。必要になった場合は `subtitle-translation` プロジェクトへの統合、あるいは連携（whisperai が出力した SRT を subtitle-translation 側で翻訳する）で対処する。

## 理由

- 字幕翻訳という関心事は既に `subtitle-translation` プロジェクトが担っており、機能の重複・二重管理を避けたい
- whisperai は「動画から SRT を作る」ことに責務を絞り、翻訳パイプラインは専用プロジェクトに委ねる方が保守しやすい

## 影響

- 現状の制約（字幕なし動画では `--translate` が無視される）は仕様として維持する
- 日本語字幕が必要な場合は、whisperai で英語 SRT を生成 → `subtitle-translation` で翻訳、という二段構成を案内する
