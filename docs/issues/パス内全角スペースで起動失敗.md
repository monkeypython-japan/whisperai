# 動画パスに全角スペースが混入していると起動に失敗する

## 症状

```
can't open file 'whisperai.py　/path/to/video.mp4': [Errno 2] No such file or directory
```

## 原因

日本語入力（IME）がオンのままスペースキーを押すと、半角スペースの代わりに全角スペース（`　`）が入力される。シェルはこれを引数の区切りとして認識しないため、コマンドとパスが1つの引数として解釈されてしまう。

## 回避策

- コマンド入力前に英数入力モードに切り替える
- パス全体をダブルクォートで囲む（全角スペースが紛れ込んでいても実害が出にくい）

## 備考

アプリ側での自動検出・警告表示は未実装（改善の余地あり）。
